Implemented sws integration protocol and fallback shell. Monitor main loop detects sws at 0x1000 and enters dispatch loop (RC>=256 = run request via shared buffer, RC 0 = quit) or falls back to built-in shell with list/help/run-by-name commands. Fixed critical UART RX bug: changed I/O pointers from int* to char* so tc24r generates lb/lbu byte reads that trigger emulator auto-acknowledge, enabling interactive input. Added string helpers and shared memory for sws interface. All commands verified via emulator UART input.
